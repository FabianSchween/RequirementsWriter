# Requirements Writer Backend

Minimales Backend-Grundgerüst fuer den MVP.

## Start

```bash
cd Backend
npm start
```

## Verfuegbare Endpunkte

- `GET /api/health`
- `GET /api/projects`
- `POST /api/projects`
- `GET /api/requirements`
- `POST /api/requirements`

## Hinweis

Aktuell ist das Frontend noch lokal mit `localStorage` verdrahtet. Das Backend ist bereits vorbereitet, damit wir im naechsten Schritt echte Persistenz und API-Anbindung sauber anschliessen koennen.
